# blumblumshub
A PseudoRandom Number Generator based on Miller-Rabin and BBS algorithm 

Personal project for the fun :D 

## How to use ? 

```bash
$ python bbs_generator.py [max_number_prime] [lenght_in_bits_random_numbers] [nb_numbers_to_generate] 
```
